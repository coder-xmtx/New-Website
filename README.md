# ✨ New-Website

该仓库用来重构个人网站，目前处于学习阶段 <br />
本网站的计划技术栈为 `Vue3` + `Pinia` + `TailwindCSS` + `VitePress` <br />
尝试使用 `TypeScript` （相当于 `JavaScript` 的超集）

## ⚙️ 项目部署说明

### 1.安装相应 node 依赖

请确保电脑已安装 NodeJS（推荐 LTS 版本）<br />
当然，用 NVM 来管理 NodeJS 版本也是个不错的选择

```sh
npm install
```

### 2.运行项目

```sh
npm run dev
```

### 3.停止项目运行

在终端里使用 `Ctrl + C`

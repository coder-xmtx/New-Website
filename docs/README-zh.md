## 安装

安装依赖：

```bash
pnpm install
```

## 开发

在 `http://localhost:3000` 启动开发服务器：

```bash
pnpm dev
```

## 构建

为生产环境构建应用程序：

```bash
pnpm build
```

本地预览生产版本：

```bash
pnpm preview
```
